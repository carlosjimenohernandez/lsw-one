# lsw-tester

Test management for LSW.

## Installation

```sh
npm i -s @allnulled/lsw-tester
```

## Importation

In node.js:

```js
require("@allnulled/lsw-tester");
```

In html:

```html
<script src="node_modules/@allnulled/lsw-tester/lsw-tester.js"></script>
```
